How It Works

Potentiometer (0–1023) maps to PWM duty cycle (0–255). Rotating the knob smoothly dims or brightens the LED using analogWrite().

Arduino Code
// Task 2: PWM Brightness Control // Author: Joyal Varghese int potPin=A0,ledPin=9; int potValue=0,brightness=0; void setup(){pinMode(ledPin,OUTPUT);Serial.begin(9600);} void loop(){ potValue=analogRead(potPin); brightness=map(potValue,0,1023,0,255); analogWrite(ledPin,brightness); Serial.print("Brightness: "); Serial.println(brightness); delay(10); }

Objective
Task Brief
Design an advanced security system combining keypad-based password authentication with PIR motion detection. Arms and disarms via a 4-digit PIN. Triggers a timed buzzer alarm with flashing LED on motion detected while armed — replicating real-world intruder alarm control panels.
Components

PIR Motion Sensor

Breadboard
Logic Flow
1
Enter 4-digit PIN (1234) on keypad
2
Press # → toggles ARMED / DISARMED
3
Press * → clears current input
4
ARMED: PIR scans continuously for motion
5
Motion → Buzzer 1kHz + flashing Red LED (5s)

Arduino Code
// Task 8: Keypad Password Security Alarm // Author: Joyal Varghese #include <Keypad.h> const String PASSWORD="1234"; String inputCode=""; const byte ROWS=4,COLS=4; char keys[ROWS][COLS]={ {'1','2','3','A'},{'4','5','6','B'}, {'7','8','9','C'},{'*','0','#','D'}}; byte rowPins[ROWS]={2,3,4,5}; byte colPins[COLS]={6,7,8,9}; Keypad keypad=Keypad(makeKeymap(keys), rowPins,colPins,ROWS,COLS); const int PIR_PIN=A0,BUZZER_PIN=10; const int LED_RED=11,LED_GREEN=12; bool armed=false,alarmActive=false; unsigned long alarmStart=0; const unsigned long ALARM_DURATION=5000; void setup(){ Serial.begin(9600); pinMode(PIR_PIN,INPUT); pinMode(BUZZER_PIN,OUTPUT); pinMode(LED_RED,OUTPUT); pinMode(LED_GREEN,OUTPUT); setStatus(false); Serial.println("=== Security System ==="); Serial.println("Enter 1234 + # to arm/disarm"); } void loop(){ char key=keypad.getKey(); if(key){ if(key=='#'){ if(inputCode==PASSWORD){ armed=!armed; alarmActive=false; noTone(BUZZER_PIN); setStatus(armed); Serial.println(armed?"ARMED":"DISARMED"); } else { Serial.println("Wrong password!"); tone(BUZZER_PIN,400,300); } inputCode=""; } else if(key=='*') inputCode=""; else inputCode+=key; } if(armed&&!alarmActive&&digitalRead(PIR_PIN)) triggerAlarm("MOTION DETECTED"); if(alarmActive){ if(millis()-alarmStart<ALARM_DURATION){ tone(BUZZER_PIN,1000); digitalWrite(LED_RED,(millis()/200)%2); } else { alarmActive=false; noTone(BUZZER_PIN); setStatus(armed); } } } void triggerAlarm(String r){ alarmActive=true; alarmStart=millis(); Serial.print("!!! ALARM: ");Serial.println(r); } void setStatus(bool a){ digitalWrite(LED_RED,a); digitalWrite(LED_GREEN,!a); }
