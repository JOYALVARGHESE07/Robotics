Interactive Output Control Tasks

Task 1: Basic Output Control (Example-Based)

Design a simple system that controls an output device.
Example: LED control, indicator system, warning light.
Think about where such systems are used in real life.

Task 2: Variable Control System

Design a system where the output behavior changes gradually or dynamically.
Example: brightness control, speed control, intensity variation.
Relate it to a real-life application such as smart lighting or automation.

Task 3: User Interaction System

Design a system that responds to user input.
Example: button-based control, switch-based logic, manual override.
Think about how humans interact with machines in daily life.
